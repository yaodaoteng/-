## 2. 快速幂

n 为非负整数

```c++
const int mod  = 1e9+7;
int pow(int x, int n){
    if(n == 0) return 1;
    int ret = pow(x, n/2);
    ret = (1LL *ret *ret)%mod;
    if(n&1) ret *= x;
    return ret;
}
```

```c++
int pow2(int x, int n){
    int ret = 1;
    int base = x;
    for(; n; n>>=1){
        if(n &1) ret = (1LL*ret*ret)%mod;
        base = (1LL *base*base)%mod;
    }
    return ret;
}
```





## 3.前缀与差分

在实现前缀与差分时， 习惯将下标从1开始

- 一维前缀和     **leetcode724**
  - 预处理 O(N):   s[i] = s[i-1] + a[i]
  - 求区间和O(1):s[r] - s[l-1]
- 二维前缀和    **leetcode304**
  - 预处理O(N * M):  s[r] [c] = s [r-1] [c] + s[r] [c-1] + a[r] [c] - s[r-1] [c-1]
  - 求子矩阵和O(1):s[r2] [c2] - s[r1-1] [c2] - s[r2] [c1 - 1] + s[r1-1] [c1-1]

- 一维差分     [P2367 语文成绩 - 洛谷 | 计算机科学教育新生态 (luogu.com.cn)](https://www.luogu.com.cn/problem/P2367)

- 预处理O(N):d[i] = a[i] - a[i-1]
- 以差分数组*d*恢复原数组a:
  - 递推恢复 O(N)  a[i] = a[i-1] + d[i]
- 区间更新操作（区间[l,  r]内每个元素都增加x）:O(1)
  - d[l] += x
  - d[r+1] -= x

## 8.单调栈单调队列

- 它们其实都是在处理窗口内的问题
- 讲到窗口，就难免涉及到窗口的伸缩问题，常见的伸缩方式有：
  - 窗口两端往同一个方向移动（但移动速度可能不同，窗口大小也是动态变化的）
  - 窗口两端往两个相反方向扩展（体现为窗口总是在变大）
  - 窗口中只有一端在扩展
- 滑动窗口相关的问题，一般是考虑窗口内的所有元素
- 单调栈&队列，一般是考虑窗口内的子序列，这个子序列具有单调性（递增/递减）
- 单调栈&队列，其实恰好对应窗口的伸缩方向
